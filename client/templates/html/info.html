Clock
This page contains a timer which displays how much time is remaining in any given period, but defaults to the current period in the school's schedule. Clicking on an arrow will scroll through the schedule for that day. If you go to a future period, the timer displays how much time remains until that period starts. Under the timer is an information board where important information will be listed.
Clubs
This page displays semi-current data on clubs ordered alphabetically.
Calendar
This page contains a color-coded calendar that shows each schedule for every day of the year. Clicking on a day will display its schedule.
